# Solidity Syntax Highlighting for Notepad++

Dark-themed **Notepad++ UDL** for Solidity (`.sol`) files. Highlights keywords, types, functions, operators, comments, and units.

## Install

1. Open **Notepad++ → Language → Define your language… → Import…**
2. Download `solidity.xml` and import it
3. Restart Notepad++ (optional)
4. Open a `.sol` file → **Language → Solidity**

## Usage

- `.sol` files get highlighted automatically if extension recognized.
- Customize colors or add keywords in the XML under **Define your language**.

## Notes

- Compatible with **Notepad++ v8.x+**
- Designed for dark theme, VSCode-like style.
